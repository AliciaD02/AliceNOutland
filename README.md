# Features
- Create a new order
- Add burgers
- Choose buns
- Add toppings and sauces
- Add drinks
- Add fries
- Checkout and view receipt
- Save receipt to a text file\


## Technologies Used
- Java
- IntelliJ IDEA
- Object-Oriented Programming (OOP)
- ArrayLists
- File I/O

## Project Structure
- `Burger.java` - Stores burger information
- `Drink.java` - Stores drink information
- `Fries.java` - Stores fries information
- `Order.java` - Stores the entire order
- `UserInterface.java` - Handles menus and user input
- `ReceiptWriter.java` - Saves receipts to a file

## How to Run
1. Open the project in IntelliJ IDEA.
2. Run the `Application` class.
3. Follow the menu prompts to create an order.
4. Checkout to view and save a receipt.
